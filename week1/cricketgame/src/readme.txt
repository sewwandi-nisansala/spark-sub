How to Play
-----------------------------------------------------------------------

Run GamePlay.java file

Enter the first team name
Press Enter

Enter the first team name
Press Enter

Now you can start the game.

Press 'p' or 'P' + Enter to hit a run(batting)


------------------------------------------------------------------------
Assumptions
------------------------------------------------------------------------

If the batsman scored 0,2,4,6 : the two batsmen are not switched
If the batsman scored 1,3 : the two batsmen are switched
If the batsman scored 5 : This is considered as a catch
If the batsman scored 7 : This is considered as a wicket fall
After finishing an over, the batsmen are switched.
Catch player and bowling player are randomly generated.
